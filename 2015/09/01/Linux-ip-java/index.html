获取本机ip的方法

一般情况下,可以直接通过InetAddress.getLocalHost().getHostAddress()获取本机ip地址.
linux中,如果hosts文件中有主机名和ip的映射,那么这个方法会返回映射的ip.等同于hostname -i返回的ip.
如果这个ip不是真实的ip,只能用NetworkInterface.getNetworkInterfaces()方法,根据ipconfig获取ip列表,从中选取一个
